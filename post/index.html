The Sass Way

Who is Sass?

Sass is a stylesheet language designed by Hampton Catlin and Natalie Weizenbaum. Originally it was developed as a feature of the HAML markup language. However, since then it has grown into it's feature rich front end tool.

What is Sass?

Sass at it's most basic definition, is a modified form of CSS that allows you to extend the funcationality of CSS. It allows you to do things most "programming languages" allow you to (e.g., create variables and functions). You can also store blocks of code in a resuable format called a "mixin". You can also partial out individual CSS files and combine them into one.
